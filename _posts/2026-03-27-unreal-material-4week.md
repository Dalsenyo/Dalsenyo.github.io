## 2 Sided (양면 렌더링)

언리얼은 기본적으로 단면 렌더링을 사용하지만 `2 Sided` 옵션을 켜면 양면 렌더링이 가능하다.

- 맥스나 유니티는 양면 지정 후 셰이더값도 따로 조절해야 라이팅이 제대로 받아지는 반면, 언리얼은 기본적으로 양면 렌더링을 잘 지원한다.
- 다만 양면도 결국 연산을 두 번 거치는 것이므로 불필요하게 쓰지 않는 게 좋다.

> **활용 예시**: IF 문으로 `2 Sided Sign` 값을 받아와서 앞면과 뒷면에 다른 색을 적용하는 양면 색종이 머터리얼 구현 가능

## Material Attributes (머터리얼 어트리뷰트)

머터리얼 에디터에서 `Use Material Attributes` 옵션을 켜면 기존의 여러 개의 큰 입력 슬롯이 하나의 작은 슬롯으로 합쳐진다.

### 왜 쓰는가?

여러 가지 재질을 하나의 머터리얼 안에 담아두고 섞거나 선택해서 쓸 수 있기 때문이다.

**활용 예시:**
- 알파값으로 높이를 기준으로 재질 1 / 재질 2 전환
- 버텍스 컬러로 조건을 넣어서 칠한 곳에만 특정 재질 적용
- 땅의 풀과 흙을 붓으로 칠해서 섞는 레이어드 머터리얼
- 이펙트 마법진 등에도 활용 가능

> `Make Material Attributes` 노드를 쓰면 여러 재질 정보를 하나로 묶을 수 있고, `Set Material Attributes` 로 원하는 값(BaseColor, Normal 등)만 골라서 세팅할 수 있다.

> Attributes를 켜지 않으면 LERP로 텍스처를 하나하나 블렌드해줘야 해서 노드가 복잡해진다.

## Alpha 파라미터 & Vertex Color 블렌딩

### Alpha 슬라이더

Alpha를 ScalarParameter로 만들어 0~1 범위의 슬라이더를 만들면 머터리얼 인스턴스에서 유기적으로 조절 가능하다.

> ⚠️ 최소값/최대값을 설정해두지 않으면 1을 초과하는 값이 들어갈 때 맵이 배수로 적용되어 이상하게 렌더링될 수 있다.

### Vertex Color를 이용한 블렌딩

버텍스 컬러의 R 값을 Alpha에 연결하면 언리얼의 **Mesh Paint** 기능으로 브러시를 사용해 직접 재질을 칠할 수 있다.

- 기본값이 1이기 때문에 검은색을 칠해야 원래 재질로 바뀐다.
- `Step` 노드를 이용하면 중간 값 없이 0 또는 1만 출력되도록 만들 수 있다 (경계가 뚜렷한 전환).

## TexCoordinate & UV 조작

언리얼의 UV 좌표계는 맥스, 블렌더와 달리 **오른쪽 아래 방향이 (1, 1)**이다.

### UV 기본 연산

| 연산 | UV에서의 의미 |
|------|--------------|
| **더하기 (Add)** | UV 오프셋 (텍스처 이동) |
| **곱하기 (Multiply)** | UV 스케일 (타일링 조절) |

### TexCoordinate 활용

- `TexCoordinate` 노드에서 `Mask`로 R 또는 G 채널만 뽑아 가로/세로 방향을 분리해서 조작 가능
- `ScalarParameter`로 타일링 수치를 파라미터화하면 머터리얼 인스턴스에서 실시간 조절 가능
- 오프셋은 Add 노드로 동일하게 구현 (기본값 0)

## Material Function (머터리얼 함수)

반복적으로 사용하는 노드 묶음을 **함수**로 만들어두면 재사용이 편리하다.

> 셰이더는 모든 버텍스에 공통으로 적용되는 수식 집합이라고 볼 수 있다. Material Function은 이 수식의 일부를 모듈화한 것이다.

### 언리얼 에셋 명명 규칙 (관례)

| 접두사 | 에셋 종류 |
|--------|----------|
| `M_` | 머터리얼 |
| `MI_` | 머터리얼 인스턴스 |
| `MF_` | 머터리얼 함수 |
| `T_` | 텍스처 |
| `NS_` | 나이아가라 시스템 |

### Material Function 만들기

1. 콘텐츠 브라우저에서 우클릭 → `Material Function` 생성
2. 함수 내부에서 노드 구성
3. `Function Input` 노드로 외부에서 값을 받을 수 있음 (Scalar, Vector, Texture 등)
4. `Expose to Library` 옵션을 켜고 `User Exposed Caption`을 설정하면 머터리얼 에디터 우클릭 메뉴에서 바로 검색 가능

> ⚠️ Function 내부에서 파라미터를 쓰면 아무 값도 들어가지 않았을 때 에러가 날 수 있다. Function Input을 사용해야 안전하다.

### 타일링/오프셋 함수 예시

TexCoordinate + Multiply + Add 구조를 Material Function으로 만들어두면 어떤 머터리얼에서든 간단하게 꺼내서 타일링과 오프셋을 조절할 수 있다.
